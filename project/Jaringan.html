Konfigurasi Jaringan Lokal Menggunakan Mikrotik

Deskripsi Proyek: Saya bersama tim mengembangkan sistem jaringan lokal berbasis MikroTik sebagai bagian dari proyek UAS Praktikum Jaringan Komputer. Proyek ini bertujuan untuk menciptakan jaringan yang stabil, aman, dan terkontrol.

Fitur Utama:

  • Routing & NAT untuk koneksi internet efisien
  • Konfigurasi DHCP Server untuk alokasi IP otomatis
  • Manajemen bandwidth menggunakan Simple Queue
  • Firewall NAT Masquerade untuk keamanan jaringan
  • Uji konektivitas & monitoring trafik

Tools yang Digunakan:

  • MikroTik RouterOS: Untuk konfigurasi jaringan secara keseluruhan melalui Winbox dan Terminal.
  • Winbox: Sebagai alat bantu visual dalam mengatur setting router MikroTik.
  • Topologi Star: Dirancang dalam bentuk star topology untuk kemudahan maintenance dan skalabilitas.
